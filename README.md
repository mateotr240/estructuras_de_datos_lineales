# Ejercicios
4. Defina de manera recursiva una función que, dados dos números enteros positivos,
calcule su multiplicación utilizando únicamente sumas repetidas. tenga en cuenta que
puede suceder si alguno de los números es 0.
• Indique claramente el caso base y el caso recursivo.
• Escriba la definición formal recursiva.
• Implemente la función en el lenguaje de programación de su preferencia, de manera
que retorne el producto de los dos números.
Entrada: a = 4, b = 3
Salida: 12"
5. Defina de manera recursiva una función que reciba como parámetro un arreglo y un
numero n, la función debe retornar la suma de los primeros n elementos.
• Indique claramente el caso base y el caso recursivo.
• Escriba la definición formal recursiva.
• Implemente la función en el lenguaje de programación de su preferencia.
Entrada: f(3, [2, 4, 6, 8, 10, 12])
Salida: 24
6. Defina de manera recursiva una función que reciba como parámetro un número entero y
retornar la suma de los primeros n números pares.
• Indique claramente el caso base y el caso recursivo.
• Escriba la definición formal recursiva.
• Implemente la función en el lenguaje de programación de su preferencia.
Entrada: 10
Salida: 30, por que 10+8+6+4+2 = 30
Entrada: 5
Salida: 6, por que 4 + 2 = 6
